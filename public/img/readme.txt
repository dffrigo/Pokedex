## Installation

Windows based
- JavaScript (Vanilla)
- Node.js v16.14.0.
- HTML
- EJS
- CSS

Requirements:
```sh
npm init
npm i nodemon --save-dev
npm i express --save
npm i --save ejs

EXPRESS START TEMPLATE:
const express = require("express");
const app = express();

app.get("/", function (req, res) {
  res.send("Hello World");
});

app.listen(3000);

```
